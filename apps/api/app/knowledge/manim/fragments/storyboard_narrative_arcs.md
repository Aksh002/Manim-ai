# Storyboard Narrative Arcs

Use this guidance before code. The generated video is educational cinema: every beat should teach
one idea and move the viewer from confusion toward understanding.

Choose one narrative arc:

- Discovery: hook, intuition, formalize, reveal, extend.
- Problem-solution: problem, failed attempt, key insight, solution, result.
- Comparison: setup, approach A, approach B, contrast, verdict.
- Build-up: component A, component B, connection, scale, full picture.
- Wrong to right: common misconception, why it fails, improved idea, correct model.
- Specific to general: concrete example, pattern, abstraction, new application.

Required storyboard fields:

- Core visual thesis in one sentence.
- Audience and assumed prerequisites.
- Misconception or motivating question.
- Hook and aha moment.
- Visual metaphor.
- Semantic color assignments.
- Three to seven visual beats with purpose, before/after state, motion intent, duration, and transition.
- Final takeaway.

Planning rules:

- Prefer geometric or spatial intuition before formulas.
- Reveal complexity gradually.
- Transform objects when possible so the viewer sees relationships.
- Give key reveals breathing room.
- Avoid disconnected slide-like scenes.
